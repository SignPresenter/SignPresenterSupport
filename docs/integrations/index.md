# Integrations

<div className="article-intro">

Integrations bring outside content into SignPresenter — Google calendars, weather data, RSS news feeds, partner curricula, your existing PowerPoint decks. Pick the integration that fits and follow the steps.

</div>

## Available integrations

- **[Google Calendar](./google-calendar)** — Pipe your public Google calendar into a Calendar Daily Overview message
- **[Lessons.church](./lessons-church)** — Free, open-source curriculum that auto-fills your weekly playlists
- **[Calendar feeds for digital signage](./calendar-feeds)** — Use Google calendars for room directories and current-event displays
- **[Show Website](./show-website)** — Embed any public web page as a message
- **[Slideshows (PowerPoint / Keynote)](./slideshows)** — Bring your existing presentation decks into SignPresenter
- **[Weather](./weather)** — Live local weather based on a zip code

## Other partners

For chiropractic clinics see [ChiroThinTV setup](../screens/chirothintv). For churches looking for sermon-style curated content, see the [Lessons.church integration](./lessons-church) above. For automatic third-party feeds in general, see [Subscribing to Feeds](../feeds/).
