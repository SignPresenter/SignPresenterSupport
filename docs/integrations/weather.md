# Weather

<div className="article-intro">

The **Weather** message pulls live local forecasts based on a US zip code. Two fields, click save, and you have an always-current weather widget on your screen.

</div>

## Setup

1. In SignPresenter, click **Messages**, then **+ New** and choose the **Weather** template — or open the default Weather message that's created with every new account.

   ![New weather message](https://user-images.githubusercontent.com/65249159/159907823-a46f174d-89f6-415d-9916-05a690f19077.png)

2. Fill in two fields:
   - **Postal Code** — used to look up the forecast
   - **City Name** — what's displayed on the weather card (this can be any label you want — "Downtown", "Main Campus", etc.)

   ![Weather fields](https://user-images.githubusercontent.com/65249159/159908624-71f7e49e-e2db-4dfd-9238-649b2474c638.png)

3. Click the green **Save** button.

   ![Save weather](https://user-images.githubusercontent.com/65249159/159908897-b2b7cb23-cc4b-4a94-be6a-35be2022b99d.png)

That's it — the weather message pulls live forecast data automatically every day.

## Tips

:::tip
Want different weather on different screens? Create a separate Weather message for each location with its own postal code, then put the right one in each screen's playlist.
:::

:::note
The forecast updates automatically — there's nothing to refresh.
:::
