# Add a Google Calendar to a Playlist

<div className="article-intro">

The simplest way to put your Google calendar on a SignPresenter screen is to use the **Calendar — Upcoming** template (or **Calendar — Daily Overview** / **Calendar — Current Event**) and point it at your calendar's public iCal URL.

## Step 1: Make your Google calendar public

1. Open Google Calendar.
2. Create a calendar (or use an existing one).
3. Add events as usual. **Tip:** if you'll use SignPresenter's room-directory features, set the event location to the room name as it appears in SignPresenter.
4. Under **My Calendars**, hover over the calendar name and click the three-dot menu. Choose **Settings and sharing**.
5. Under **Access permissions for events**, check **Make available to public** and click **OK**.

## Step 2: Copy the iCal URL

1. Scroll down the settings page.
2. Find **Public address in iCal format**.
3. Copy the URL.

## Step 3: Paste into SignPresenter

1. Return to SignPresenter and create a new **Calendar Upcoming** message (or Daily Overview, or Current Event).
2. Paste the iCal URL into the **ICS URL** field.
3. Click **Save**.

The calendar updates automatically — any change you make in Google Calendar syncs to SignPresenter on the next refresh.
