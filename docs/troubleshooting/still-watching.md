# Disable the "Still Watching?" Pop-up on Fire TV

<div className="article-intro">

Starting with Fire OS 7, Amazon devices show a "Still Watching?" dialog after 4 hours of inactivity. If the dialog isn't dismissed, the device sleeps. For 24/7 signage, you'll want this off.

</div>

:::note
This pop-up is generated by Fire OS itself, not by SignPresenter. It started appearing by default in a 2023 Fire OS update.
:::

## What the pop-up looks like

![Still Watching pop-up](https://github.com/SignPresenter/SignPresenterSupport/assets/65249159/15389b2a-ffa6-4e42-8f8e-d9232129b9f5)

## Quick fix: dismiss it once

If the pop-up is currently on screen, press **"Don't show again"**. That deactivates the feature and saves the setting.

## Permanent fix: disable in system settings

1. From the Fire TV home screen, click the **gear icon** to open system settings.
2. Select **Preferences**.

   ![Preferences](https://github.com/SignPresenter/SignPresenterSupport/assets/65249159/306a43c8-5189-4a19-8902-1df06e7d3d50)

3. Find and click **Data Usage Monitoring**.

   ![Data Usage Monitoring](https://github.com/SignPresenter/SignPresenterSupport/assets/65249159/75ecbf97-da0d-4523-8b44-f6686baa9ebb)

4. Locate **Still Watching** and turn it **Off**.

   ![Still Watching off](https://github.com/SignPresenter/SignPresenterSupport/assets/65249159/65b294e6-fd4e-48a5-990a-c081eaeb689f)

The pop-up should never appear again, and your display will run uninterrupted.
