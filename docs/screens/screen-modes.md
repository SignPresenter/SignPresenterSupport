# Screen Modes

<div className="article-intro">

SignPresenter screens can run in three modes: **Digital Signage** (auto-rotating), **Slideshow** (manually advanced), or **Both**. Picking the right mode depends on whether someone is actively presenting or whether the content runs unattended.

## When to use each mode

| Mode | Use it when… |
|------|--------------|
| **Digital Signage** | The screen runs unattended on a schedule — lobbies, menu boards, announcement walls. |
| **Slideshow** | Someone is presenting in person and wants to advance slides manually with the remote. |
| **Both** | You want both behaviors — for example, a classroom that shows a normal signage rotation between lessons but switches to a manual slideshow during teaching. |

## Setting the mode

1. Go to **Step 3: Screens** and select your screen.
2. In the **Mode** dropdown, choose **Digital Signage**, **Slideshow**, or **Both**.
3. Click **Save**.

When set to **Both**, your Fire Stick remote's **Back** button toggles between the scheduled signage and the manual slideshow. The scheduled signage uses the playlists in the **Schedules** box; the slideshow uses the **Manual Playlist** dropdown.
