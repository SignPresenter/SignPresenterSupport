# Getting Started

<div className="article-intro">

Three steps from zero to "my screens are showing my content": create your first message, build a playlist, and pair a TV. This section walks through each one.

</div>

## What you need

- A free SignPresenter account at [signpresenter.com](https://signpresenter.com)
- A Fire TV Stick or Android device running Android 10+ (see [Compatible Devices](../screens/compatible-devices))
- A TV with an HDMI input

## Start here

1. **[Initial Setup Walkthrough](./initial-setup)** — the full three-step setup with video
2. **[Create Your First Message](./first-message)** — pick a template, fill it in, save
3. **[Build a Playlist](../playlists/)** — sequence your messages
4. **[Pair Your Screen](./initial-setup#step-3-connect-your-screen)** — install the app and enter the 4-digit code

:::tip
We recommend Google Chrome for editing in SignPresenter — other browsers occasionally have video preview quirks.
:::
