## Messages

A piece of content displayed on your screen is called a **message**. Click the **Messages** tab to see your library — anything in here can be added to a playlist.

To create a new image message:

1. Click the green **+ New** button.
2. Select the **Simple Image** content type.
3. Fill out the fields on the left:
   - **Duration** — how long the message should be shown
   - **Category** — a folder name to keep your messages organized
   - **Name** — a display name for the message
4. Click the image thumbnail to launch the image editor.
5. Click the green **+ Upload Image** button and browse to your image.
6. If desired, use the cropper to select the portion of the image you want to display, then click **Update**.
7. Click **Save**.

:::note
Creating a message just adds it to your library — it isn't shown on a screen yet. The next step is adding it to a [playlist](../playlists/).
:::

## Playlists

A **playlist** is a collection of messages. A default "Custom Playlist" is created automatically and contains the weather message. Add your new message to it, or create additional playlists if you want different content at different times.

The Custom Playlist is already scheduled to run on your screen, so once your message is in it you can preview by going to the **Screens** tab and clicking **Preview**. Once it looks good, you're ready to [pair a TV](../screens/) and start displaying.
