# Messages

<div className="article-intro">

A **message** is anything you display on a screen — an image, a video, a designer graphic, a weather widget, or a live web page. Messages are the building blocks of every playlist.

## Recommended dimensions

| Ratio | Pixels | Use case |
|-------|--------|----------|
| 16:9 | 1920 × 1080 | Landscape full-screen |
| 9:16 | 1080 × 1920 | Vertical signage |
| 4:1 | 1280 × 320 | Marquee / message strip in a [three-zone layout](../playlists/three-zone-messages) |

## Message templates

SignPresenter ships with templates for every common signage need. Pick the one that fits, fill in the fields, and save.

| Template | What it does |
|----------|--------------|
| **Simple Image** | Display a single image on a timed loop |
| **Video With Audio** | Play a video with sound — see [Uploading Videos](./uploading-videos) |
| **Silent Video** | Same as above without audio |
| **Video 1080p / Video 4K** | Higher-resolution video (4K only on a 4K Stick with plenty of storage) |
| **Designer** | Build custom graphics in the browser — see [Digital Menus](./digital-menus) |
| **Image Gallery** | Upload multiple images at once and rotate through them |
| **Weather** | Live weather pulled from your zip code — see [Weather](../integrations/weather) |
| **Show Website** | Embed a live web page — see [Show Website](../integrations/show-website) |
| **Pdf Document** | We recommend converting PDFs to PNG and using Simple Image |
| **Pointer** | A "pointer" message used by the [departments](../account/departments) workflow |
| **News Feed (RSS)** | A scrolling RSS news ticker |
| **Trivia** | Built-in trivia questions on rotation |
| **Three Zone** | Combine three independent zones into one screen — see [Three-Zone Messages](../playlists/three-zone-messages) |
| **HTML** | Drop in raw HTML for full control |
| **Live Video** | Stream live video sources |
| **Calendar — Daily Overview** | Day-at-a-glance view from a Google calendar |
| **Calendar — Upcoming** | Upcoming events from a Google calendar |
| **Calendar — Current Event** | What's happening right now (great for room signs) |
| **Clock** | Digital or analog clock |
| **Canva** | Pull live designs from your Canva account — see [Canva](./canva) |
