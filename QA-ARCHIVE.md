## Technical Support

**Q: Why does my screen preview say it's not supported?**
**A:** If you're unable to preview videos on SignPresenter.com, the issue may be related to your browser. We recommend using Google Chrome for the best experience, but even then, you might encounter an error. A common issue is the following: Error: NotAllowedError: play() failed because the user didn't interact with the document first. This occurs due to Chrome's security restrictions, which prevent videos from autoplaying in the preview popup. To resolve this, click anywhere in the preview popup while it's loading. This action bypasses the restriction. Over time, as you continue using SignPresenter, this workaround typically won't be necessary because your usage builds up in the browser's history. Just note that this usually works fine on devices like Fire Sticks, even if the browser preview fails.

To configure and set it up, you may need to manually switch the input on the TV between the computer and the SignPresenter device when needed, but the best approach would be to manually switch the input on the TV between the computer and the Fire TV Stick. Some TVs can be set to automatically switch between HDMI inputs, but that depends on your TV's specific features. If you'd prefer full automation, a professional AV setup with HDMI switchers and triggers could work, though that may be more complex than needed. For a low-cost solution that avoids compatibility issues with content that isn't suited for SignPresenter, the best approach would be to manually switch the input on the TV between the computer and the SignPresenter device when needed.

## Office Calendar Integration

**Q: Can I use SignPresenter to set up a TV as an office calendar?**
**A:** Yes, SignPresenter offers a great way to display an office calendar on a TV, showing upcoming events, current events, and more. Here's how to set it up: Getting Started with Office Calendar Display: Step 1: Message tab in the top-left corner of SignPresenter. Under the calendar section, select Directory. Use the Calendar Daily Overview to see a list of events with times, event names, and rooms for the day. Use the Calendar Current Events to display what's happening now, synced with Google Calendar. Use the Upcoming Calendar Overview to show upcoming events, pulled directly from Google Calendar. Customizing Calendar Display: Change Text Color and update the background. Click the background image icon in the Appearance box. In the Crop box, click Upload in the top-right corner. Select an image from your computer, crop it to fit, and click Save. To change text color: Search for "HTML Color Code" online to find a color code (e.g., via a color picker tool). Paste the code into the Text Color box and click Save. Syncing with Google Calendar: Open Google Calendar in your browser. Create a new calendar, adding event names, times, dates, etc. Set up schedules. In SignPresenter, go to Settings and Sharing. Under Access Permissions, check Make available to public and click OK. Scroll to Public address in iCal format, click the iCal-formatted address. In SignPresenter, paste the iCal link into the Calendar field (under Messages â†’ Directory or Current Events) to link your calendar. Select Refresh Time (5 minutes to 1 hour). Choose the number of days to sync (1 to 30). To display all schedules: Set the link to Calendar Current Events and Calendar Daily Overview to sync all views. Any changes made in Google Calendar will automatically sync with SignPresenter, updating all calendar views. For a step-by-step guide, watch this video: https://www.youtube.com/watch?v=Y-tD9Z2lnfuojU
